# Blog_mini

### 1.功能强大的开源博客系统
Blog_mini是一个开源的博客系统，用Python开发完成，具有简洁的界面和强大的后台管理，

使用它，你可以轻松架设属于您自己的个人博客网站！

    只要把Blog_mini部署上，您不需要再进行任何命令上的操作即可以轻松使用和管理您的个人博客！
